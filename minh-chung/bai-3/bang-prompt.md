# Bài 3: Bảng Prompt học tập

## Tác vụ 1: Tóm tắt tài liệu học thuật

- Prompt cơ bản: "Tóm tắt tài liệu sau bằng tiếng Việt."
- Prompt cải tiến: "Tóm tắt tài liệu theo 5 ý chính, nêu thuật ngữ quan trọng và kết luận trong 150 từ."
- Prompt nâng cao: "Đóng vai trợ giảng Hệ thống thông tin, tóm tắt theo bảng: luận điểm, bằng chứng, ví dụ, câu hỏi cần ôn."

## Tác vụ 2: Giải thích khái niệm phức tạp

- Prompt cơ bản: "Giải thích khái niệm dữ liệu lớn."
- Prompt cải tiến: "Giải thích dữ liệu lớn cho sinh viên năm nhất, có ví dụ trong thương mại điện tử."
- Prompt nâng cao: "Đóng vai chuyên gia dữ liệu, giải thích theo mô hình: định nghĩa, đặc điểm, ví dụ, lỗi hiểu sai, câu hỏi kiểm tra."

## Tác vụ 3: Tạo câu hỏi ôn tập

- Prompt cơ bản: "Tạo câu hỏi ôn tập về AI."
- Prompt cải tiến: "Tạo 10 câu hỏi trắc nghiệm và 5 câu tự luận về AI trong học tập, kèm đáp án."
- Prompt nâng cao: "Tạo bộ câu hỏi theo 3 mức: nhớ, hiểu, vận dụng; kèm đáp án, lỗi thường gặp và gợi ý tự học."

## Kết luận

Prompt càng có ngữ cảnh, vai trò, định dạng đầu ra và tiêu chí đánh giá thì kết quả càng cụ thể, dễ kiểm tra và dễ sử dụng hơn trong học tập.
